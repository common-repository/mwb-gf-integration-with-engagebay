== Description ==

**MWB GF INTEGRATION WITH ENGAGEBAY PROVIDES A SMOOTH CONNECTION OF GRAVITY FORMS WITH ENGAGEBAY. SEND ALL YOUR DATA (SUBMISSIONS/ENTRIES) AUTOMATICALLY OVER THE ENGAGEBAY.**

MWB GF Integration with EngageBay unites the GF with EngageBay. This plugin sends all the data (form submissions/entries) over the EngageBay as per the modules present in the EngageBay. It is one of the best plugins for gravity forms engagebay integration.

Gravity Form is one of the best WordPress plugins that permit admins, developers, and store owners to create forms for collecting crucial information. On the other hand, EngageBay is All-in-One sales, marketing and service software that provides powerful tools for business growth. 

But Gravity Forms doesn’t provide the functionality to send its submissions/entries over EngageBay. This plugin is a combination of these efficient tools- Gravity Forms and EngageBay. With the help of this plugin, you can send all your Gravity Forms submissions/entries to EngageBay in real-time.

== WHAT IS INCLUDED IN MWB GF INTEGRATION WITH ENGAGEBAY PLUGIN? ==

**1.** **Integrate Gravity Forms With EngageBay**
In order to connect Gravity Forms with EngageBay enter your API credentials (API Key, Base URL). View your number of synced forms, connection status, owner name, etc. on the plugin’s dashboard.

**2.** **Associate Form Fields With EngageBay Modules**
Create Gravity Forms and associate form fields with EngageBay modules. Map form fields with respective object fields. Mapped fields will be sent over to EngageBay on the form submission. Easily sync all your Gravity Form submissions data over EngageBay.

**3.** **Filter GF Submissions As Per The Feeds**
You can filter the Gravity Form submissions based on the user input using AND/OR logic. The form submission data will be sent over EngageBay only if it fulfils the conditional logic condition.

**4.** **Get Detailed Logs**
This plugin provides comprehensive logs. It will display all the information regarding the data sent over EngageBay via Gravity Forms such as associated form, object and mapped fields, response, timestamp, etc.

**5.** **Delete Entire Data Upon Plugin Uninstallation**
MWB GF Integration with EngageBay allows the users to delete all the data from the database upon the plugin uninstallation.

**6.** **Receive Error Email Notifications**
Admins have the choice to enable/disable error email notifications. They will receive error email notifications whenever any error occurs while sending submission data over EngageBay.

**7.** **EngageBay Custom Values**
You can choose a custom value for field mapping of the field from the field type. Your custom value will sync. Also, you can name the field.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of the gift voucher, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type **"MWB GF Integration with EngageBay"** and click Search Plugins. Once you've found our plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking **"Install Now"**.

= Manual installation =

The manual installation method involves downloading our MWB GF Integration with EngageBay and uploading it to your webserver via your favorite FTP application. The WordPress contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).

= Does this plugin provide comprehensive logs? =
